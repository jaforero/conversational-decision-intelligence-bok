# Candidate v0.6.0-rc.1

`v0.6.0-rc.1` begins Sprint 4—**Depth and learning experience**. It addresses a specific gap: the portal contained strong definitions and governance but did not clearly state what readers would gain or provide a complete application path.

## What it adds

- a home page centered on framing, evaluating, designing, governing and learning;
- navigation by decision friction as well as taxonomy;
- five connected modules from Decision First to learning;
- a one-page Decision Brief template;
- responsive components and sprint-specific controls;
- ADR-020 and content-map traceability.

## What it does not change

The `v0.4.0` core retains authority; the B2B case remains instrumented and unexecuted; no organizational-impact evidence is added; completing the path does not guarantee better outcomes.

## PR exit criteria

Owner editorial review, strict build, semantic audit, automated accessibility in both themes, and approved Home and Learning Path visual baselines were completed before merge. These gates do not prove effectiveness or stable promotion.
