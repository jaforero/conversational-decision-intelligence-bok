# Release v0.8.1

`v0.8.1` establece el **portal bilingüe ES/EN como línea base editorial y técnica estable**. Promueve `v0.8.1-rc.1` después de comprobar 46 pares de contenido, navegación e interfaz localizadas, rutas equivalentes, metadatos SEO y controles automáticos contra deriva.

## Alcance estable

- español canónico en las rutas históricas e inglés completo bajo `/en/`;
- selector contextual que conserva la página equivalente;
- navegación, búsqueda, interfaz, anuncios, footer y metadata social localizados;
- canonical, `hreflang` ES/EN y `x-default`;
- registro de traducciones con versión, estado, owner, revisión y hash de la fuente;
- build estricto, auditoría semántica, Playwright, regresión visual y Axe.

## Linaje

El PR #12 implementó el candidato bilingüe y se fusionó como `655e3b80ad1cbd8b443b6339358996f9fe656083`. El PR #13 corrigió los residuos españoles de la interfaz inglesa y se fusionó como `5837657f6bfe0627f5bd0c8ada13640dba9a0a4b`. La validación post-hotfix `29882684997`, el despliegue `29882684960` y la comprobación pública HTTP 200 cerraron el candidato antes de esta promoción.

## Límites

- El inglés es traducción gobernada; no crea doctrina co-canónica.
- El núcleo `v0.4.0` conserva su autoridad aprobada.
- Aprendizaje, medición y patrones conservan madurez candidata.
- El caso B2B permanece instrumentado y no ejecutado.
- La estabilidad del portal no valida científicamente CDI o PULSE ni demuestra eficacia o impacto.
- Los controles automatizados no establecen conformidad completa con WCAG 2.2 AA.

ADR-026 autoriza el tag y GitHub Release `v0.8.1` únicamente sobre el SHA de `main` que supere los gates estables posteriores al merge.
