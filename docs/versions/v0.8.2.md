# Release v0.8.2

`v0.8.2` convierte los ajustes posteriores a `v0.8.1` en una línea base estable. La edición española localiza sus rótulos visibles, la portada explica mejor el resultado para usuarios de negocio y la búsqueda respeta de forma estricta el idioma seleccionado.

## Qué cambia

- Navegación y títulos españoles localizados, con el término controlado inglés en la primera aparición.
- Tooltip `Enlace permanente` en ES y `Permanent link` en EN.
- Propuesta de valor de portada centrada en mejores decisiones de negocio.
- Índices ES/EN separados, cada uno con 47 páginas raíz y sin rutas del otro idioma.
- Controles automáticos para impedir regresiones de localización o mezcla de corpus.

## Qué no cambia

- La autoridad aprobada sigue limitada al núcleo `v0.4.0` y a los registros que correspondan.
- Aprendizaje, medición y patrones siguen siendo candidatos.
- PULSE no se presenta como científicamente validado.
- El caso B2B continúa instrumentado y no ejecutado.
- Un build estable no demuestra impacto organizacional.

ADR-027 autoriza el tag y GitHub Release `v0.8.2` únicamente sobre el SHA de `main` que supere todos los gates posteriores al merge.
