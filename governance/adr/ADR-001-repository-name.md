# ADR-001: Nombre del repositorio oficial

## Decisión

El repositorio oficial se denomina `conversational-decision-intelligence-bok`. El nombre privilegia claridad y descubribilidad sobre brevedad. El portal público mantiene la denominación editorial `CDI-BoK`.

## Consecuencias

Las URLs técnicas, badges y automatizaciones usarán el nombre largo; la navegación y comunicación usarán CDI-BoK. Un cambio posterior exige redirecciones y un nuevo ADR.
