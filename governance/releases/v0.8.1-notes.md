# CDI-BoK v0.8.1 — Portal bilingüe ES/EN estable

`v0.8.1` promueve el portal bilingüe a una línea base editorial y técnica estable. Publica 46 rutas españolas canónicas y 46 ediciones inglesas equivalentes, con interfaz, búsqueda, navegación y metadatos localizados.

## Incluye

- selector ES/EN que conserva la página equivalente;
- canonical, `hreflang` ES/EN y `x-default`;
- registro de traducciones con hashes de fuente y gate de obsolescencia;
- interfaz inglesa completa, incluido Learn, Practice Lab, tema y footer;
- build estricto, auditoría semántica, Playwright, regresión visual y Axe;
- linaje verificable de los PR #12 y #13.

## Fronteras

- El español permanece canónico durante `0.x`.
- El inglés no crea doctrina independiente.
- La autoridad aprobada del núcleo `v0.4.0` se preserva.
- Aprendizaje, medición y patrones continúan como candidatos.
- El caso B2B permanece instrumentado y no ejecutado.
- La release no demuestra validación científica, eficacia, adopción o impacto.

ADR-026 gobierna esta promoción y autoriza `v0.8.1` únicamente sobre el SHA validado de `main`.
